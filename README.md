 MemePic Memory Match

MemePic Memory Match is a card-matching memory game where players flip cards to find matching images before time runs out.  
Players can choose a difficulty level and track their time and number of moves.
---
Game Features
- Three difficulty levels: Easy, Medium, Hard  
- Timer and move counter  
- Sound effects and background music  
- Animated card flip effects  
- Image-based memory cards  
---
Technologies Used
- HTML5 – Game structure  
- CSS3 – Styling, animations, and layout  
- JavaScript (Vanilla JS) – Game logic and interactivity
